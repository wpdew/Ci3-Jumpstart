This includes setting up: * authentication * templating engine * custom controllers * default template In the pipeline will be: * factories (for generating data for testing) * testing suite

Features

  • Built with CodeIgniter 3.1
  • HMVC architecture
  • Secure user registration and login
  • Password reset
  • Secure password algorithm using Bcrypt
  • Avatar and logo upload with crop feature
  • Remember Me feature on login
  • Login with email or username
  • CAPTCHA on registration
  • E-Mail Verification for new users
  • Interactive Dashboard
  • Powerful admin panel
  • Manage permissions
  • Assign permission to users
  • Easy installation using installation wizard
  • Built using AdminLTE
  • Client side and server side form validation
  • Fully object oriented

Server Requirements

  • PHP version 5.4 or newer is recommended.
  • MySQL (5.1+) via the mysqli

Installation

Старт.чч..

  • Duplicate en.php (default language) file and rename it to your desired lanauge name i.e. es.php
  • Now translate the sentences to your language in es.php.

For example; Translate the term or word on right side of "=" inside quotes

$lang['code']                   = 'es';
$lang['language']               = 'Español';
$lang['direction']              = 'ltr';
$lang['date_format_short']      = 'd/m/Y';
$lang['date_format_long']       = 'l dS F Y';
$lang['time_format']            = 'h:i:s A';
<p>Введите   сюда код, символы которого необходимо преобразовать, нажмите "преобразовать" - результат появится в окне ниже.<p>
<?php
  echo"asdsa";
?>

TO

$lang['code']                   = 'es';
$lang['language']               = 'Español';
$lang['direction']              = 'ltr';
$lang['date_format_short']      = 'd/m/Y';
$lang['date_format_long']       = 'l dS F Y';
$lang['time_format']            = 'h:i:s A';

Now goto setting and you will see new language in drop down. Select the language you want to set default and save.
